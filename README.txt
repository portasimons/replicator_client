Основной файл - "replicator.py":
Для отправки http-запроса необходимы соединение по TCP и рабочий http-сервер.
Чтобы обеспечить тестовое соединение, нужно запустить файлы "TCP server.py" и "http-server.py".
Если сервера недоступны, программа "крутится" в вечном цикле и ждет, пока появится 
соединение.
В консоли http-сервера будут отображаться все запросы отправленные основной программой.

Файл конфигурации "config.txt":
Поля 1 и 2 - адрес для TCP-сервера
Поле 3 - адрес HTTP-сервера
Поле 4 - путь к файлу базы данных
Поле 5 - индекс последнего rowid
Поле 6 - таймаут проверки соединения с TCP-сервером
Поле 7 - таймаут проверки базы данных
